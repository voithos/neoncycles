# NeonCycles

![NeonCycles](https://raw.github.com/voithos/neoncycles/main/NeonCycles.gif)

Light up the path.

NeonCycles is a prototype Tron-inspired arcade survival game built in 24 hours,
in which your only means of offense is your neon trail. But be careful, it's
dangerous to yourself as well!

## Controls

Player one:

- Turn - left / right arrow keys
- Speed up - up arrow
- Slow down - down arrow
- Esc to exit

Player two:

- Turn - A / D keys
- Speed up - W key
- Slow down - S key
